HRMS Project

Human Resource Management System (HRMS)

This project is developed using the MERN Stack.

Features

- Employee Management
- Attendance Tracking
- Leave Management
- User Authentication
- Dashboard and Reports

Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js

Project Objective

The HRMS application helps organizations manage employee information, attendance, leaves, and other HR-related activities efficiently.
